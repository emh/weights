Workout List

This is the main screen. Each row shows the workout date, day of week, volume/intensity marker, and exercise summary.

  • Tap a row to expand or collapse its exercise details.
  • Tap the pencil on the current row to open that workout for editing.
  • Tap delete to remove a workout.
  • The circle icon is a visual indicator of relative volume and intensity. Color indicates intensity: yellow = low, orange = moderate, red = high and gray means there wasn't sufficient data to calculate intensity. The circle size indicates volume: small, medium, large. These will only show up once there is enough data.
  • Click the + button to create a new workout.

Create Workout

This is the inline add-workout flow on the main screen. Enter the date first, then optionally add exercises as a comma-separated list.

  • Tap the + button to create a new workout.
  • Enter the date of the workout like 2026.03.08
  • Optionally include a comma-separated list of exercises like: 2026.03.08 bench press, deadlift.

View Workout

View workout details on main screen.

  • Tap a workout to make it current and see its details.
  • Tap the pencil to edit the workout.
  • Tap the delete button to remove the workout.

Edit Workout

Edit the workout details on the workout screen.

  • Enter a new exercise into the exercise input field. Optionally use the sets/reps/weight shorthand to include sets and reps like: "bench press 5@135 165 5x5@195".
  • Tap an exercise to make it current and expand its sets. Add additional sets in the Sets input field.
  • Tap the pencil to edit the exercise name and details.
  • Tap the delete button to remove the exercise from the workout.
  • Tap on a set to edit the reps and weight and to set an RPE.
  • Tap the pencil on the current exercise to edit it. Here you can use shorthand like: "5x5@225 3@255" to quickly enter sets and reps.
  • Use the notes field to add or edit notes for the workout.

History

The history drawer gives context for the currently selected exercise inside a workout.

  • Tap an exercise to select it and reveal the history drawer.
  • Tap the history drawer to expand it and see past efforts for that exercise as well as a list of maxes at various reps.
  • Tap a max to scroll to that entry in the history.

Settings

The gear icon on the main screen shows the settings panel.

  • Export CSV will download all your workout data as a CSV file.
  • Import CSV will load workout data from a saved CSV file.
  • Font size and Theme Color adjust the appearance of the app.
  • Delete all workout data will remove all your saved workouts.

Entry Shorthand

You can enter workouts, exercises, and sets in a compact text format. These are the most useful patterns.

  • New workout: 2026.03.08 bench press, deadlift, row
  • Date shortcuts also work: t for today, y for yesterday, tmr for tomorrow.
  • New exercise with sets: bench press 5x5@225
  • Mixed sets: deadlift 5@315 5@365 5@405
  • Repeated weighted sets: 3x5@225
  • Repeated unweighted sets: 4x8
  • Timed and distance sets: plank 3x30s, sled push 4x20m@90, carry 3x60ft@70/70
  • Split values are supported: kb snatch 3x6/6@24kg, side extension 2x10/10
  • After one weighted set, @ reuses the previous load: 5@225 5@ 5@
  • You can separate set tokens with spaces or commas: 5@225 5@245 5@265 or 5@225, 5@245, 5@265
